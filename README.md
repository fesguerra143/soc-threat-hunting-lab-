# SOC Threat Hunting Investigation Lab Projects
MITRE ATT&CK Based Threat Analysis

## Project Overview

These projects demonstrate threat hunting investigation using Microsoft Defender for Endpoint telemetry, Sentinel Log Analytics and the MITRE ATT&CK framework to identify suspicious network activity within an enterprise environment.

The investigation analyzes abnormal connection failures originating from an internal host and evaluates whether the activity aligns with known adversary behaviors.

## Objectives

• Identify suspicious network behavior  
• Investigate device communication patterns  
• Map activity to MITRE ATT&CK techniques  
• Develop KQL queries for detection  

## Tools Used

- Microsoft Defender for Endpoint
- KQL (Kusto Query Language)
- MITRE ATT&CK Framework
- Microsoft Sentinel Log Analytics

## Investigation Workflow

1. Identify abnormal network activity
2. Run threat hunting queries
3. Analyze device communications
4. Correlate suspicious events
5. Map findings to MITRE ATT&CK
